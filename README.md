HOW TO RUN IT? -->>

first install go from official wesite;

then install msys2 from msys2.org enable gcc;

then open msys2 and type "pacman -S mingw-w64-ucrt-x86_64-gcc";

then copy all these file to a folder;

then open cmd of the folder and type "go run main.py"
